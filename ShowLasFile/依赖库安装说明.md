# ShowLasFile 项目依赖库安装说明

## 需要的库

### 1. libLAS
LAS点云文件读取库

**安装方法：**
- 下载：https://liblas.org/
- 或使用vcpkg：`vcpkg install liblas`

### 2. PCL (Point Cloud Library)
点云处理库

**安装方法：**
- 下载：https://github.com/PointCloudLibrary/pcl/releases
- 或使用vcpkg：`vcpkg install pcl`

### 3. Boost
C++扩展库（PCL依赖）

**安装方法：**
- 下载：https://www.boost.org/
- 或使用vcpkg：`vcpkg install boost`

## .pro文件配置

需要在ShowLasFile.pro中添加库路径：

```qmake
# libLAS
INCLUDEPATH += "C:/path/to/liblas/include"
LIBS += -L"C:/path/to/liblas/lib" -llas

# PCL
INCLUDEPATH += "C:/path/to/pcl/include"
LIBS += -L"C:/path/to/pcl/lib" -lpcl_common -lpcl_octree

# Boost
INCLUDEPATH += "C:/path/to/boost/include"
LIBS += -L"C:/path/to/boost/lib" -lboost_system
```

## 使用vcpkg安装（推荐）

```bash
# 1. 安装vcpkg
git clone https://github.com/Microsoft/vcpkg.git
cd vcpkg
./bootstrap-vcpkg.bat

# 2. 安装依赖
./vcpkg install liblas:x64-windows
./vcpkg install pcl:x64-windows

# 3. 集成到Qt Creator
./vcpkg integrate install
```
